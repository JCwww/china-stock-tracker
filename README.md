# 中国股票跟踪表

一个部署在 GitHub Pages 上的中国 A 股跟踪网页，支持按股票代码增删查，并展示：

- 股票名称
- 股票代码
- 添加起始日价
- 起始日起最高价
- 每日收盘价
- 涨幅
- 最高价回撤
- 起始价回撤

公开列表在 `data/stocks.json` 中维护，GitHub Actions 会在工作日 16:10 中国时间附近自动更新 `data/market.json`。页面内新增或删除的股票会保存在当前浏览器本地。

## 公式

```text
涨幅 = (起始日起最高价 - 添加起始日价) / 添加起始日价
最高价回撤 = (起始日起最高价 - 每日收盘价) / 起始日起最高价
起始价回撤 = ((添加起始日价 - 每日收盘价) / 添加起始日价) / -1
```

## 更新公开列表

修改 `data/stocks.json` 后提交到 `main` 分支，等待 `Update market data` workflow 运行，或在 GitHub Actions 页面手动运行它。
